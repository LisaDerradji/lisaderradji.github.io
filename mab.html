MY ART BLOG

est un site web spécialisé dans l'art, l'architecture et le design. Le thème est basé sur le plugin javascript masonry qui affiche les listes d'articles et d'images sous forme de grille. Le design du site est entièrement responsive et optimisé pour un chargement rapide, même sur les plus petits appareils.

La création propremement graphique comprend :

  • le logo d'entête
  • la bare de navigation
  • l'intégration de ces éléments dans le code du site

Logiciels utilisés :

  • Adobe Illustrator CC 2015
  • Adobe Photoshop CS6
  • Wordpress v.4.8.2
